# targets

A collection of fuzzing targets written in Rust.

## How do I fuzz?

As an example:

```sh
./run-fuzzer.sh mp4parse read_mp4
```

## Contributing

Want to add another fuzz target? It can be for an existing crate or a new one, just open a pull request!
